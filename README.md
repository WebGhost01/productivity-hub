# Productivity Hub

All-in-one hub with 7 apps:

- **Job Tracker** — Job application pipeline
- **Link Vault** — Bookmark manager with collections
- **Habit Chain** — Daily habit streaks tracker
- **Event Board** — Event planning & RSVP
- **Study Deck** — Flashcard app with spaced repetition
- **AI Assistant** — Streaming GPT-5.2 chat + image generation
- **Space Shooter** — Canvas Space Invaders game

## Stack
React 19 · TypeScript · Tailwind CSS · Vite · Framer Motion · OpenAI API

## Run
```bash
npm install && npm run dev
```
Requires [portfolio-api-server](https://github.com/WebGhost01/portfolio-api-server).
